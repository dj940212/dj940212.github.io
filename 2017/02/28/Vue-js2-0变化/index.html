Vue.js2.0变化

2017-02-28
  • 编程

模版

  • 组件模版，不支持片段代码，必须要有一个跟元素，包裹住所有模版代码（1.0支持）

组件定义

  • Vue.component(组件名称，{ //vue2.0继续存在
    data(){}
    methods:{}
    tmplate:
    })

  • vue2.0推出了简洁的组件定义方式
    var name = {
    template:’’
    };

生命周期

beforeCreate -> 组件实例刚刚被创建，属性尚未绑定
created -> 实例已经创建完成，属性已经绑定
beforeMount -> 模版编译之前
mounted -> 模版编译完成
beforeUpdate -> 组件更新之前
updated -> 组件更新完毕
beforeDestroy -> 组件销毁之前
destroyed -> 组件销毁完毕

循环

默认就可以添加重复数据（vue1.0需设置 track-by=”$index”）

过滤器

去掉了很多系统自带过滤器(limitBy,filterBy,debounce,currency)，自定义过滤器传参改成函数方式 number | toDou(a,b)

父子组件数据传递

vue1.0: 子组件可以更改父组件数据，可以使父组件同步（sync）
vue2.0: 不允许直接改数据
如何更改：

1.父组件每次传一个对象给子组件，数据可作为对象属性，然后可以更改

2.可以使用mounted更改
