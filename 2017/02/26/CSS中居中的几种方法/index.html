CSS中居中的几种方法

发表于 2017-02-26

本文主要总结一下平常所能用到的各种使用CSS居中的方法。

水平居中

宽度未知

margin: auto 0水平居中

平常比较常用的方法，前提是不受float影响

text-align：center水平居中

img的display：inline-block；类似一样在不受float影响下进行
实在父元素上添加效果让它进行水平居中

水平垂直居中

宽高已知

绝对定位和margin设为负值

这种方法的局限性在于需要知道需要垂直居中的宽高才能实现，经常使用这种方法

宽高未知

绝对定位和margin:auto

绝对定位和transfrom

diplay：table-cell水平垂直居中

其实这个就是把其变成表格样式，再利用表格的样式来进行居中

flexBox水平垂直居中(兼容IE11以上)

这个是CSS3的新特性
